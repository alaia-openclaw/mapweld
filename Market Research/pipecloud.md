# Competitor deep-dive — PipeCloud

URL: https://pipecloud.fi/

## What they sell (positioning)
- Cloud MES for **pipe prefabrication** (planning → shopfloor execution → tracking).
- Strong outcome messaging: productivity + waste reduction + automation.

## Key pages
- Overview/home: https://pipecloud.fi/
- Features (welding traceability): https://pipecloud.fi/features/welding-traceability/
- FAQ (ICP/pricing signals): https://pipecloud.fi/faq/

## ICP clues (from FAQ)
Source: https://pipecloud.fi/faq/
- Roles explicitly listed: production planners, fitters, welders, cutting/bending/logistics operators, quality inspectors, production managers.
- Sells to small workshops **and** large shipyards.
- Integrations: API + ERP/scheduling; CAD compatibility.
- Pricing: “based on feature package + number of user licenses; tailored quote.”
- Cadence: “weekly updates” (signals rapid product iteration and SaaS expectations).

## Welding traceability framing (useful copy cues)
Source: https://pipecloud.fi/features/welding-traceability/
- Positions traceability as essential in regulated industries: naval, LNG, offshore, nuclear.
- Specific data they highlight capturing:
  - welder name, weld type, WPS used
  - filler material + batch, consumption
  - arc time, voltage, current
- Outputs: traceability reports auto-generated in **PDF + Excel**.
- Explicit integration mention: “PipeCloud also integrates with WeldEye by Kemppi”.

## How they win (hypothesis)
- When buyer wants **full MES**: planning, bundling, CNC programs, capacity planning.

## MapWeld implications
- PipeCloud is a strong “outcome competitor” if the prospect wants a full shop-wide operating system.
- Best response is the “layer” story:
  - MapWeld = **drawing/evidence/pack layer** that can run alongside ERP/MES.
  - Focus on ISO/PDF reality + rapid inspector adoption.
