# Objection Handling (Pilot Motion)

## "We already have spreadsheets."

Response:
"That is normal. The pilot does not replace your spreadsheet immediately. It reduces retyping by capturing traceability on the drawing first, then exporting for office reporting."

## "We need multi-user cloud features."

Response:
"This pilot validates workflow value first on one project. If value is proven, we can scope broader deployment requirements in phase two."

## "We do not have time to onboard."

Response:
"Kickoff is one focused session. Activation requires loading one PDF, creating 10+ records, and testing save/export. Most teams complete this in under one hour."

## "How do we trust data durability?"

Response:
"Pilot process includes explicit save/reopen validation with `.weldproject` files and export checks in day-1 activation."

## "Can this replace our ERP/MES?"

Response:
"Not directly. MapWeld is a capture layer for field and QA traceability, with export paths for existing enterprise systems."
