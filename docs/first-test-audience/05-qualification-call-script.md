# Qualification Call Script (20 minutes)

## Objective

Confirm the prospect is a strong pilot candidate and align on one-project pilot scope.

## Agenda (time-boxed)

1. Context and goals (3 min)
2. Current workflow and pain points (6 min)
3. Fit check to pilot scope (5 min)
4. Pilot logistics and next steps (6 min)

## Talk track

### 1) Context and goals

- "We run a focused 30-day pilot on one active project."
- "Goal is to prove faster, cleaner weld traceability from drawing to export."

### 2) Discovery questions

- "How are welds tracked today from field/shop to office register?"
- "Where does reconciliation break most often?"
- "How many weld records do you handle per week on a typical job?"
- "Who owns final handover output?"
- "What does a failed handover usually look like?"

### 3) Fit criteria check

- Must have active project PDF.
- Must have a named champion.
- Must have weekly 20-minute review availability.
- Must accept one-project pilot scope and anonymized feedback capture.

### 4) Close and next steps

- If fit:
  - Confirm kickoff date.
  - Confirm champion and backup contacts.
  - Share activation checklist and required inputs.
- If not fit:
  - Park for later phase and capture blocker category.

## Qualification scoring

Use a simple score from 0 to 8 (2 points each):

- Active project now.
- Clear traceability pain.
- Champion ownership.
- Review cadence commitment.

Proceed when score is 6+.
