# Social Clip Production Pack (15-30 seconds)

## Goal

Create a short, hook-first clip that drives replies and pilot call clicks on LinkedIn and short-form channels.

## Technical specs

- Duration target: 22 seconds.
- Format: 1080x1920 vertical (primary), plus 1080x1080 square backup.
- Audio: optional voiceover; must work with captions muted autoplay.

## Hook options (choose one)

- "Your weld register is already out of date."
- "Stop tracking welds in disconnected spreadsheets."
- "If it is not on the drawing, it is not traceable."

## 22-second script and shot list

| Time | Visual | Voiceover | Overlay text |
|------|--------|-----------|--------------|
| 0-3s | Fast cut: spreadsheet + PDF + chat screenshot | "Your weld register is already out of date." | `Your weld register is already out of date.` |
| 3-8s | Load PDF in MapWeld | "Open the isometric and track directly on it." | `Open PDF. Track on drawing.` |
| 8-13s | Add 2-3 weld markers quickly | "Place welds and capture the required fields as you go." | `Place welds. Capture traceability.` |
| 13-17s | Save project file | "Save one portable `.weldproject`." | `Save .weldproject` |
| 17-20s | Excel export preview | "Export to Excel when office needs it." | `Export register to Excel` |
| 20-22s | End card | "Book a 30-day pilot." | `Pilot one live project.` |

## Caption text blocks

1. Your weld register is already out of date.
2. Open the isometric and track directly on it.
3. Place welds and capture traceability fields.
4. Save one portable `.weldproject`.
5. Export to Excel when office reporting is needed.
6. Pilot one live project in 30 days.

## Platform cutdown instructions

- LinkedIn feed:
  - Keep first text line visible in first second.
  - Add static CTA comment with booking link.
- YouTube Shorts/Reels:
  - Use 3-5 hashtag tags only.
  - Keep title problem-led, not feature-led.
- X/Twitter:
  - Use square variant for better timeline framing.

## Edit checklist

- Keep total overlays under 6 text cards.
- Ensure every card is readable within 1.5-2.0 seconds.
- Remove UI micro-pauses and loading waits.
- Avoid jargon in hooks; keep terms operator-friendly.
