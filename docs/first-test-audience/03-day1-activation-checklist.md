# Day-1 Activation Checklist

Use this checklist during kickoff. Activation is complete only when all required items are done.

## Pre-kickoff (required)

- [ ] Champion assigned.
- [ ] Pilot project selected.
- [ ] Project PDF available and anonymized.
- [ ] Expected export columns confirmed.

## In-app setup (required)

- [ ] Open `/app`.
- [ ] Load pilot PDF.
- [ ] Confirm drawing is visible and scroll/zoom works.
- [ ] Add at least 10 weld markers.
- [ ] Fill required weld fields on at least 5 records.
- [ ] Link at least one spool or part where relevant.

## Save and restore (required)

- [ ] Save a `.weldproject` file locally.
- [ ] Re-open the saved `.weldproject`.
- [ ] Confirm weld markers and key metadata are restored.

## Reporting (required)

- [ ] Export Excel register.
- [ ] Validate headers and row count against expected output.
- [ ] Share one sample output file in pilot channel.

## Quality check (recommended)

- [ ] Open Health panel.
- [ ] Review top 3 findings.
- [ ] Decide owner for each finding (shop, QA/QC, office).

## Activation pass/fail

- Pass if all required sections are complete in one session.
- Fail if save/restore or export is blocked; escalate within 24 hours.
