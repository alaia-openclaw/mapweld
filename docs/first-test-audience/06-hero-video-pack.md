# Hero Video Production Pack (60-90 seconds)

## Goal

Show a believable end-to-end MapWeld workflow for mixed teams and drive pilot call bookings.

## Technical specs

- Duration target: 75 seconds.
- Format: 1920x1080 landscape (master), export 1080p.
- Version outputs:
  - Full hero (60-90s) for website/YouTube.
  - 45s cut for LinkedIn feed.
- Style: clean UI capture, no exaggerated effects, clear cursor movement.

## Narrative structure

1. Problem framing (0-10s)
2. Workflow proof (10-55s)
3. Output and confidence (55-68s)
4. CTA (68-75s)

## Voiceover script (full)

"Weld tracking breaks when drawings and spreadsheets drift apart. Most teams still retype updates across tools.  
In MapWeld, open your isometric and work directly on it. Place welds, capture traceability fields, and keep context on the drawing.  
Link spools and parts without leaving the workspace. Switch to inspection mode when you need controlled review.  
Save one portable `.weldproject` file your team can reopen. Export a weld register when office reporting is needed.  
Run health checks before turnover to catch missing links early.  
Start a guided 30-day pilot and validate on one real project."
