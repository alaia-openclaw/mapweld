### ASME B16.5 Flanges — view filter and submenu

#### Main toolbar (detail view)

- **Left:** Component type **dropdown** + **first / previous / next / last** (step through types).
- **Center:** **Subtype-dependent** parameter dropdowns (see table below). Examples use DN80 / Class 600# where captured.
- **Right:** **Search**, **Print**, **Export**, **Settings**, **Help**.

#### Center filters by flange subtype (not identical for all types)

| Flange type | Parameters on blue bar (typical) |
|-------------|----------------------------------|
| Weldneck Flange | Size, Class, Face Type, **Schedule** (e.g. 80s) |
| Slip-on Flange | Size, Class, Face Type _(no Schedule on bar)_ |
| Blind Flange | Size, Class, Face Type |
| Threaded Flange | Size, Class, Face Type |
| SocketWelded Flange | Size, Class, Face Type |
| Lapped Flange | Size, Class, **Schedule** _(Face Type not on bar in this view)_ |
| Straight Hub Welding Flange | Size, Class, Face Type |

#### Flange type submenu (dropdown list)

1. Weldneck Flange
2. Slip-on Flange
3. Blind Flange
4. Threaded Flange
5. SocketWelded Flange
6. Lapped Flange
7. Straight Hub Welding Flange

#### Detail view content (typical)

- **Title line:** e.g. `DN80 Weldneck Flange 600# RF Sch-80s` (wording varies by subtype; Lapped example: `DN80 Lapped Flange 600#`).
- **Weights:** flange mass, studs/bolts & nuts (when shown).
- **Drawing** + dimensions; **footer:** stud bolt line (e.g. UNC/M size × length), standard **`ASME B16.5-2020`** on the right.
