- **Center:** **Size** (DN), **Class** (e.g. 150# … 2500#; captures include **600#**). No separate **Face Type** dropdown on the bar — **RF** (raised face) appears in the title.

#### Center filters (all three lines)

| Line | Parameters on blue bar |
|------|-------------------------|
| B16.5 | Size, Class |
| B16.47 Series A | Size, Class |
| B16.47 Series B | Size, Class |

#### Detail view content (typical)

- **Title line:** `DN{size} Nonmetallic Flat Gasket for ASME … Flanges {class} RF` (e.g. `DN40 … 600# RF`, `DN650 … 600# RF`).
- **Weight:** single mass line (e.g. **0.02 kg** for DN40 / 600# B16.5; **0.85 kg** for DN650 / 600# on B16.47 A & B captures).
- **Drawing:** cross-section — flange / bolt / nut assembly; gasket as thin layer between faces. Callouts: **Client specified** (gasket thickness), **Self centering** (outer edge vs bolt circle). Horizontal dimensions for **inner** and **outer** gasket diameters (examples from captures: B16.5 DN40 600# → **48** / **95** mm; B16.47 Series A DN650 600# → **660** / **867** mm; B16.47 Series B DN650 600# → **660** / **765** mm).
- **Footer — right:** gasket standard **`ASME B16.21-2016`** _(not the flange B16.5 / B16.47 standard)._

#### Per-line notes

- **B16.5:** Smaller-DN example in capture: Size **40**, Class **600#**.
- **B16.47 Series A vs B:** Same nominal **Size** / **Class** can yield **different outer** gasket dimensions (e.g. DN650 600#: OD **867** mm vs **765** mm in captures) — Series A and B flange/GCD differ per B16.21 tables.

**MapWeld:** Part catalog — `Gasket → Nonmetallic Flat Gaskets →` (three standards above). Data `lib/nonmetallic-flat-gaskets-data.js`; UI `PanelCatalogNonmetallicFlatGaskets.jsx`.
